What is Npcap?
Npcap is a packet capture library and a driver for Windows. It is the modern, enhanced version of the older WinPcap library. Npcap allows software like Scapy to access raw network packets and sniff the network for analysis. It provides the underlying functionality that enables packet capture and network analysis.

Scapy, which is used in your CLI tool, depends on Npcap (or WinPcap on older systems) to perform packet sniffing. Without a packet capture driver like Npcap, Scapy cannot intercept and analyze network packets.
